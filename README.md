# 月度工作记录App

这是一个简约风格的月度工作记录应用，使用uniapp开发，支持安卓平台。

[图片示例](./static/page.png)


## 功能特点

### 📊 核心功能
- **月度工作记录**：首页显示当月总收入
- **记录收支**：支持日期选择、数量输入、单价设置（保留3位小数）
- **历史记录**：按年月筛选，显示每日详细记录
- **实时更新**：添加、删除记录后实时刷新总金额

### 📱 用户界面
- **米色毛玻璃UI**：采用现代简约设计，米色渐变背景
- **响应式布局**：适配不同屏幕尺寸
- **交互动画**：按钮点击效果和页面过渡动画

### 📈 统计功能
- **本月统计**：显示工作天数、日均收入、最高日收入、总记录数
- **智能分组**：按日期分组显示历史记录
- **实时计算**：自动计算每日和每月总金额

## 使用指南

### 快速开始
1. 打开应用后，首页显示当月总收入
2. 点击"记录收支"按钮添加新的工作记录
3. 点击"查看历史记录"查看详细的历史数据
4. 点击"本月统计"查看月度统计信息

### 记录收支
- **日期选择**：默认为今天，可自定义选择任意日期
- **数量输入**：请输入工作数量（如：小时数、项目数等）
- **单价设置**：输入单价，自动保留3位小数
- **实时计算**：自动计算并显示总计金额

### 历史记录
- **时间筛选**：可按年份和月份筛选记录
- **详细显示**：显示每一天的具体记录，包括数量、单价、总计
- **删除功能**：每条记录后都有删除按钮，支持二次确认
- **实时刷新**：删除记录后自动更新显示的总金额

## 技术特性

- **数据存储**：使用uniapp本地存储，数据安全可靠
- **响应式设计**：适配各种安卓设备
- **流畅体验**：优化的用户界面和交互设计
- **离线使用**：无需网络连接，完全离线运行

## 界面设计

应用采用米色毛玻璃设计风格：
- 主色调：米色渐变背景 (#f5f7fa → #c3cfe2)
- 卡片效果：毛玻璃卡片，半透明背景
- 按钮样式：圆角按钮，渐变色彩
- 字体配色：深色文字配合绿色金额显示
